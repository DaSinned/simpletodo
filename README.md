# SimpleTodo
Simple command line script to store tasks in a database

### possible commands:
- list (shows entries)
- done ID (remove entry with ID from database)
- add TEXT @ DATE (add a new entry)

### possible 'add' commands:
- add TEXT
- add TEXT @ 31.12.2018
- add TEXT @ 31.12.2018 15:01
- add TEXT @ WORD 
- add TEXT @ WORD 15:01
